[역할 정의]
너는 전문적인 소프트웨어 제품 프레젠테이션 제작자야. 
'Mermaid Sequence Diagram Editor'라는 웹 애플리케이션의 소개 및 사용 가이드 슬라이드를 만들어야 해.

[청중]
- 시퀀스 다이어그램을 더 쉽게 그리고 싶은 개발자, 기획자, PM.
- 코딩 없이 다이어그램을 수정하고 싶어하는 비개발자 협업자.

[발표 톤앤매너]
- 혁신적이고 모던한 느낌 (Premium Dark Mode 테마 강조)
- 간결하고 명확한 설명
- "복잡한 코드는 가라, 직관이 왔다"는 식의 자신감 있는 어조

[슬라이드 구성 요청]
총 5~7장의 슬라이드로 구성해줘. 각 슬라이드는 '제목', '핵심 메시지', '세부 내용(3~4줄)', '시각 자료 아이디어'를 포함해야 해.

[핵심 기능 소스]
아래 내용을 바탕으로 내용을 구성해:

1. **Dual Editor System**: 왼쪽엔 엑셀 같은 Grid Editor, 오른쪽엔 실시간 미리보기. 코드(Markdown)와 GUI가 완벽하게 양방향 동기화됨.
2. **One-Click Actions**: 드래그 앤 드롭으로 순서 변경, 버튼 클릭으로 화살표 모양 변경, Swap 버튼으로 방향 전환.
3. **Smart Validation**: Mermaid 문법의 까다로운 Activation(막대바) 짝 맞추기를 자동으로 해결해 줌. 실수로 끊어진 문법을 알아서 고쳐줌.
4. **Interactive Viewer**: 줌(Zoom)과 팬(Pan) 기능으로 대형 다이어그램도 부드럽게 탐색 가능.
5. **Data Privacy**: 서버로 데이터를 보내지 않고, 브라우저 로컬 스토리지에 안전하게 저장. 버전 관리(이력 복구) 기능 내장.

[출력 형식]
Markdown 형식으로 작성해 줘. 각 슬라이드는 구분선(---)으로 나눠줘.
