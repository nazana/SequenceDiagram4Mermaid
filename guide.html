튜토리얼 보기 바로 시작하기

Grid Editor

엑셀처럼 행을 추가하고 내용을 입력하면 됩니다. 복잡한 괄호나 화살표 기호는 신경 쓰지 마세요.

Dual Sync

Grid에서 수정하면 Markdown이, Markdown을 수정하면 Grid가 즉시 업데이트됩니다.

Auto Correct

실수로 꼬인 Activation(막대바) 순서를 자동으로 감지하고 올바르게 고쳐줍니다.

Dashboard

폴더로 정리하고, 단축키로 빠르게 관리하세요. 전체 화면 모드도 지원합니다.

단계별 튜토리얼

1

참여자(Participant) 정의하기

에디터를 열면 상단에 참여자 목록이 보입니다. 기본으로 생성된 Alice, Bob 등의 이름을 클릭하여 내 프로젝트에 맞게 수정하세요.

참여자 이름 클릭 -> 수정 -> 엔터

2

메시지 흐름 만들기 (Grid)

Source(보내는 사람)와 Target(받는 사람)을 선택하고 메시지를 입력합니다.

순서가 틀렸다면? 각 행 왼쪽의 핸들(::)을 잡고 드래그하여 위아래로 옮기세요.

Tip: Swap () 버튼을 누르면 보내는 사람과 받는 사람이 즉시 뒤바뀝니다. 응답 메시지를 만들 때 유용합니다.
3

활성화(Activation) 제어

메시지를 보낼 때 생명선을 활성화하려면 (+) 버튼을 누르세요. 작업이 끝나는 시점에 (-) 버튼을 누르면 비활성화됩니다.

초록색(+)은 활성화 시작, 빨간색(-)은 종료

4

줌 & 팬으로 탐색하기

다이어그램이 커지면 마우스 휠로 줌(Zoom)을, 빈 공간을 드래그하여 팬(Pan)을 할 수 있습니다.

꿀팁: 단축키와 관리

대시보드 단축키

대시보드에서 카드의 ... 메뉴를 열면 다음 단축키를 사용할 수 있습니다.

  • M : 이동 (Move)
  • D : 복제 (Duplicate)
  • Space : 상세 편집
  • Del : 삭제

전체 화면 모드 (Fullscreen)

대시보드 메뉴에서 Fullscreen을 선택하면 다이어그램만 깔끔하게 볼 수 있습니다.

- 자동 맞춤: 화면 크기에 맞춰 다이어그램이 자동으로 줌인/아웃 됩니다.

- 나가기: 우측 상단의 X 버튼을 누르면 돌아옵니다.

자주 묻는 질문 (FAQ)

Q. 데이터는 어디에 저장되나요?

A. 100% 사용자의 브라우저 로컬 스토리지에 저장됩니다. 서버로 전송되지 않아 보안상 안전합니다.

Q. 이미지가 안 보여요 (Zoom 문제)

A. 우측 하단의 초기화 버튼()을 눌러보세요. 줌 레벨이 100%로 돌아오고 위치가 중앙으로 맞춰집니다.

Q. Markdown 탭은 언제 쓰나요?

A. Grid Editor로 표현하기 힘든 복잡한 문법(Loop, Alt, Note 등)을 직접 수정하고 싶을 때 사용하세요. 수정한 내용은 Grid에도 반영됩니다.
