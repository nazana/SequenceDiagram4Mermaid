# 변경 이력 (Changelog)

이 프로젝트의 모든 주요 변경 사항은 이 파일에 문서화됩니다.

## [Unreleased]

## [0.2.0] - 2025-12-14
### 변경됨 (Changed)
- **Mermaid 라이브러리 업데이트**: v10.9.0 -> v11.0.2 (양방향 화살표 지원).
- **화살표 문법 변경**: Open Arrow (`->`, `-->`)를 Mermaid 비동기 문법인 `-)`, `--)`로 변경하여 표준 준수.

### 추가됨 (Added)
- **화살표 타입 확장**: 총 8가지 화살표 타입 지원 (Solid/Dotted Line, Open Arrow, Cross, Bidirectional).
- **양방향 화살표 지원**: `<<->>`, `<<-->>` 문법 및 UI 지원 추가.
- **향상된 화살표 선택 UI**: 아이콘 미리보기가 포함된 팝업 형태의 선택기 구현.
- **데이터 정규화**: 입력된 Mermaid 코드의 화살표 문법을 자동으로 정규화하고 오류를 수정하는 `sanitize` 로직 추가.

### 수정됨 (Fixed)
- **UI 버그**: 다크 모드에서 Open Arrow가 보이지 않는 문제 해결.
- **렌더링 버그**: Grid Editor에서 화살표 변경 시 즉시 반영되지 않는 문제 해결.
- **초기화 오류**: 기본 예제 데이터(`DEFAULT_DIAGRAM`)의 화살표 문법을 명확한 타입(`->>`)으로 변경하여 파싱 오류 방지.
- **Grid Editor**: 양방향 화살표 아이콘의 왼쪽 마커 방향 오류 수정.

## [0.1.0] - 2025-12-13
### 추가됨
- **듀얼 에디터**: 마크다운과 그리드 에디터 간 양방향 동기화 지원.
- **Mermaid 렌더링**: Panzoom(줌/팬) 기능을 포함한 실시간 다이어그램 시각화.
- **데이터 관리**: 로컬 스토리지 기반의 저장 시스템 및 버전 이력 관리.
- **UI/UX**: 프리미엄 다크 테마, 반응형 레이아웃, 접이식 사이드바 구현.
- **대시보드**: 다이어그램 목록 조회 및 관리 기능.
